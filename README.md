# Geo Quiz Frontend
Frontend del proyecto Geo Quiz. 

Cuestionario de preguntas hecho con Angular implementando E2E testing con Cypress.

## Pasos para instalar y ejecutar el proyecto
Instalar las dependencias del proyecto
```
npm install
```
Correr el proyecto 
```
ng serve -o
```
